# 🛍️ ShopVerse - Modern E-Commerce Website

ShopVerse is a responsive and modern E-Commerce website built using **HTML, CSS, and JavaScript**. It provides an attractive user interface with interactive features such as product search, shopping cart counter, wishlist functionality, dark mode, flash sale countdown timer, newsletter subscription, and local storage support.

---

## 🚀 Features

### 🎨 User Interface
- Modern and responsive design
- Sticky navigation bar
- Hero section with call-to-action
- Product categories section
- Featured products section
- Flash sale section
- Customer reviews section
- Newsletter subscription section
- Professional footer

### ⚡ JavaScript Functionality
- Add to Cart system
- Dynamic Cart Counter
- Product Search
- Wishlist Toggle
- Dark Mode Toggle
- Flash Sale Countdown Timer
- Newsletter Validation
